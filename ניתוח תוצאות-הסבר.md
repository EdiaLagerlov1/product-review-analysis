# ניתוח תוצאות - הסבר

## סקירה כללית

מערכת זו משווה בין שני אלגוריתמי למידת מכונה:
- **K-means (למידה לא מפוקחת)**: מקבץ ביקורות לפי דמיון ללא תוויות - דיוק 82.8%

## 1. אשכולות K-means

![אשכולות K-means](outputs/run_20251104_000509/plots/01_kmeans_clusters.png)

**מה רואים בגרף:**
- 600 נקודות במרחב דו-ממדי (הורד מ-100 מימדים באמצעות PCA)

**פרשנות:**

- אשכול 1 (ירוק): בעיקר ביקורות חיובי (190 מתוך 283)
- אשכול 2 (כחול): בעיקר ביקורות שלילי (167 מתוך 170)

**מסקנה:** K-means הצליח למצוא את הדפוסים הסנטימנטליים רק מניתוח המילים!

## 2. השוואה: תוויות אמיתיות מול תחזיות K-means

![השוואת תוויות](outputs/run_20251104_000509/plots/02_original_vs_predicted.png)

**מה רואים בגרף:**
- **פאנל שמאלי**: התוויות האמיתיות (אדום=שלילי, ירוק=ניטרלי, כחול=חיובי)
- **פאנל ימני**: תחזיות K-means (אדום=אשכול 0, ירוק=אשכול 1, כחול=אשכול 2)
- אותן נקודות בדיוק, אותו מיקום במרחב

**פרשנות:**
- הצבעים שונים כי K-means נותן מספרי אשכולות (0, 1, 2) ולא תוויות סמנטיות
- האזור הכחול בשמאל (חיובי) = בעיקר ירוק בימין (אשכול 1)
- האזור הירוק בשמאל (ניטרלי) = בעיקר אדום בימין (אשכול 0)
- האזור האדום בשמאל (שלילי) = בעיקר כחול בימין (אשכול 2)

**מסקנה:** K-means מצא את הקבוצות הנכונות אבל עם מספור שונה!

## 3. מטריצת בלבול K-means

![מטריצת בלבול K-means](outputs/run_20251104_000509/plots/03_confusion_matrix_kmeans.png)

**מה רואים בגרף:**
- מטריצה 3×3 המשווה בין תוויות אמיתיות (שורות) לאשכולות שנמצאו (עמודות)
- צבעים כהים = מספרים גבוהים

**מספרים:**
```
              אשכול 0  אשכול 1  אשכול 2
שלילי              0       33      167    ← רוב באשכול 2
ניטרלי           140       60        0    ← רוב באשכול 0
חיובי              7      190        3    ← רוב באשכול 1
```

**פרשנות:**
- אשכול 0 תפס 140/147 = 95% ניטרליות
- אשכול 1 תפס 190/283 = 67% חיוביות
- אשכול 2 תפס 167/170 = 98% שלילות

**דיוק כולל:** (140+190+167)/600 = **82.8%** - מצוין ללמידה לא מפוקחת!

## 5. מטריצת בלבול KNN

![מטריצת בלבול KNN](outputs/run_20251104_000509/plots/06_confusion_matrix_knn.png)

**מה רואים בגרף:**
- מטריצה 3×3 עם אלכסון חזק מאוד
- אלכסון כהה = סיווג נכון
- מחוץ לאלכסון = שגיאות (מעט מאוד)

**מספרים:**
```
              תחזית שלילי  תחזית ניטרלי  תחזית חיובי
אמת שלילי          196            4              0
אמת ניטרלי            0          194              6
אמת חיובי             0            2            198
```

**פרשנות:**
- **שלילי**: 196/200 = 98% דיוק
- **ניטרלי**: 194/200 = 97% דיוק
- **חיובי**: 198/200 = 99% דיוק
- **אין טעויות קיצוניות**: אף פעם לא בלבל שלילי עם חיובי!

**8 השגיאות:**
- 4 שלילי → ניטרלי (שפה פחות קיצונית)
- 6 ניטרלי → חיובי (ניסוח מעט חיובי)
- 2 חיובי → ניטרלי (שפה חיובית מתונה)

## מסקנות

### 1. K-means (למידה לא מפוקחת)
**יתרונות:**
- ✅ לא דורש תוויות - מוצא דפוסים לבד
- ✅ דיוק 82.8% - טוב מאוד בלי תוויות
- ✅ מהיר - שנייה אחת
- ✅ מעולה לגילוי דפוסים לא ידועים

**חסרונות:**
- ⚠️ דיוק נמוך יותר מ-KNN
- ⚠️ מתקשה עם הקטגוריה החיובית (רק 67%)
- ⚠️ דורש התאמה ידנית של מספרי אשכולות לתוויות

### 2. KNN (למידה מפוקחת)
**יתרונות:**
- ✅ דיוק 98.67% - כמעט מושלם
- ✅ 99% דיוק בזיהוי חיוביות
- ✅ אין טעויות קיצוניות
- ✅ יציב - ולידציה צולבת 95.67%

**חסרונות:**
- ⚠️ דורש 600 דוגמאות מתוייגות
- ⚠️ תלוי באיכות התוויות

### 3. Word2Vec
- המפתח להצלחה: הבנה סמנטית של מילים
- "מצוין" ו"נהדר" קרובים במרחב 100 מימדים
- פי 27 שיפור ב-K-means לעומת TF-IDF

### 4. המלצה
- **יש לך תוויות?** → השתמש ב-KNN (98.67% דיוק)
- **אין לך תוויות?** → השתמש ב-K-means (82.8% דיוק)
- **רוצה לחקור דפוסים?** → K-means מעולה למחקר ראשוני
